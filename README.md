# Md Saifullah Portfolio

Personal portfolio website for Md Saifullah, built to present backend engineering work, featured projects, skills, and contact details in a clean single-page experience.

Live site: https://iamsaifullah.me

## Overview

This project is a React + Vite portfolio with animated sections for hero content, experience, education, skills, projects, GitHub activity, and contact details. It is designed for deployment on Vercel with a custom domain.

## Features

- Responsive portfolio layout with dark and light theme support
- Project showcase with screenshots, feature summaries, and source links
- SEO-ready metadata, sitemap, robots file, and social sharing tags
- Resume download and direct contact links
- Vercel Analytics integration

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Radix UI components

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deployment

1. Push the repository to GitHub.
2. Import the repo into Vercel.
3. Set the production domain to iamsaifullah.me in Vercel.
4. Point the domain DNS to Vercel using the records provided in the Vercel dashboard.

## Notes

- Update any personal links in the source files if your social profiles or contact details change.
- Keep the sitemap and canonical URL aligned with the production domain.
